# SortMaster — 英语学习笔记整理 Skill

> 把杂乱的 B1 级别英语课堂笔记，一键整理成 Obsidian 可用的结构化复习卡片。

---

## 这是什么

**SortMaster** 是一个 AI Skill，专为 B1 水平英语学习者设计。

把任意杂乱的课堂笔记、单词列表或学习截图丢给 AI，SortMaster 自动整理成标准 Markdown 格式，包含：

- 📋 **词汇表** — 词性、英文释义、例句、中文翻译
- 🗣️ **核心表达** — 使用场景 + 对话示例
- 💡 **语法要点** — 规则说明 + 场景速查
- 📖 **课文 / 对话** — 中英对照 + 要点注释
- 🔁 **复习题** — 含隐藏答案，支持 Obsidian 自测

---

## 安装方式

**方式一：发链接给 AI（最简单）**
把下面这个链接直接发给 Claude 或其他 AI，说"帮我安装这个 skill"：
```
https://github.com/rjmirror1117-maker/Mirror
```

**方式二：Claude 桌面版（Cowork）手动安装：**
1. 下载 `SortMaster.skill` 文件
2. 进入 **设置 → Skills**
3. 点击 **Install from file**
4. 上传文件 ✅

**方式三：Claude Code / Codex CLI / OpenCode 等命令行工具：**
将 `SortMaster.skill` 解压后放入项目的 `.claude/skills/` 目录即可。

> 兼容平台：Claude Code · Codex CLI · Cursor · Gemini CLI · OpenCode · 任何支持 Skill 的 AI 工具

---

## 使用方式

安装后直接把英语笔记丢给 AI，说：

> "帮我整理一下"
> "整理这节课的笔记"
> "整理成复习用的格式"

AI 会自动识别并触发 SortMaster，输出的 Markdown 直接复制粘贴进 Obsidian 即可用。
